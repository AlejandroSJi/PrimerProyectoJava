# PrimerProyectoJava

En este repo, anexe 3 proyectos pequeños en los que uso los conceptos basicos de JAVA, el primero es una **calculadora**, tratando de hacer un diseño mas acercado a los UI actuales, el segundo es un juego de **3 en raya**, en el cual se lleva un marcador y un boton de reiniciar, y por ultimo un juego sencillo de **Snake** el cual aumentara de dificultad conforme pasa el tiempo.
